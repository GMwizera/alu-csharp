# C# - Data Structures: HashSet, Stack, Queue, LinkedList

Methods working with C#'s `HashSet<T>`, `Stack<T>`, `Queue<T>`, and `LinkedList<T>` collections.

## Tasks

| # | Folder | Description |
|---|--------|-------------|
| 0 | `0-unique_add` | Sum of the unique integers in a list |
| 1 | `1-common_elements` | Sorted list of elements common to two lists |
| 2 | `2-different_elements` | Sorted list of elements in one list or the other but not both |
| 3 | `3-stack_push_pop` | Inspect a `Stack`, remove items up to a search term, push a new item |
| 4 | `4-queue_enqueue_dequeue` | Inspect a `Queue`, enqueue an item, remove items up to a search term |
| 5 | `5-print_linkedlist` | Create and print a `LinkedList` of a given size |
| 6 | `6-linkedlist_length` | Number of elements in a `LinkedList` |
| 7 | `7-linkedlist_add` | Add a node to the beginning of a `LinkedList` |
| 8 | `8-linkedlist_find` | Index of the first occurrence of a value, or -1 |
| 9 | `9-linkedlist_pop` | Remove the head node and return its value |
| 10 | `10-linkedlist_get_node` | Value of the node at a given index |
| 11 | `11-linkedlist_sum` | Sum of all data in a `LinkedList` |
| 12 | `12-linkedlist_insert` | Insert a node in order into an ascending `LinkedList` |
| 13 | `13-linkedlist_delete` | Delete the node at a given index |

## Running a task

```
cd <task-folder>
dotnet run
```

Each task folder contains its own `.cs` source and `.csproj`. Compiled on Ubuntu with `dotnet`.
