# C# - Structs and Enums

This project covers the basics of structs and enumeration types in C#, including
constructors, fields, properties, and overriding the `ToString()` method.

## Learning Objectives

- What is a struct
- What is a constructor
- What is a field
- What is a property
- What is an enumeration and when to use them
- What does `ToString` do and how to override it

## Tasks

| Task | File | Description |
|------|------|-------------|
| 0. They're good dogs | [0-dog/](0-dog/) | Define an enum `Rating` with the values `Good`, `Great`, `Excellent`. |
| 1. Chief Puppy Officer | [1-dog/](1-dog/) | Define a struct `Dog` with `name`, `age`, `owner`, and `rating` members. |
| 2. A dog is the only thing... | [2-dog/](2-dog/) | Add a parameterized constructor to struct `Dog`. |
| 3. A dog will teach you unconditional love | [3-dog/](3-dog/) | Override the `ToString()` method to print the `Dog` object's attributes. |

## Environment

- Compiled on Ubuntu with `dotnet`
- Each task has its own folder and `.csproj` file
