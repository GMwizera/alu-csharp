# C# - Hello, World

This project introduces the basics of C# programming using the .NET CLI on Linux/macOS. It covers initializing a new C# console application, writing and running C# code, and using compilation tools.

## Requirements

- Ubuntu 20.04 LTS or macOS
- .NET SDK 8.0 or higher
- All scripts must start with `#!/usr/bin/env bash`
- All `.cs` files should compile without errors

## Tasks

| File | Description |
|------|-------------|
| `0-initialize_new_project.sh` | Bash script that initializes a new C# console project in a folder named `0-new_project` using the `dotnet` command. |
